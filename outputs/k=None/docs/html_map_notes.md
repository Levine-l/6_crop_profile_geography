# HTML map notes

The notebook can export exploratory HTML maps under `outputs/maps`.

These maps are useful for checking:

- crop-profile cluster geography;
- selected-crop intensity;
- dominant crop proportions;
- zone-level SAR disagreement reduction;
- zone-level CROME-UKCEH disagreement.

For the dissertation, treat HTML maps as supplementary inspection tools. Do not rely on a live interactive map for assessed evidence. Export static PNG figures and tables for the main text.
